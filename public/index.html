WOWIE

WOWIE is an express app that uses Blizzard API to save information about World of Warcraft characters in MongoDB and then display it in the Characters tab.

You have option to update characters to current in game state or to delete them from data base so they won't appear in Characters list anymore.

If you wish to add more characters simply go to Character Add section, fill the form with real data and then go back to Characters tab. Your character should be at the end of the list.
